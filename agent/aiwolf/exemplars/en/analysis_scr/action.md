# Analysis — action (exemplar, from scripts)

提示されたお手本は、5人村という極めて手数の少ない環境において、「情報の透明性」と「発言の整合性」が勝敗を分けることを示しています。特にDay 0の振る舞いがDay 1の真贋判定の根拠となる点、および対抗CO（カウンタークレイム）が出た際の論理的矛盾の突き方が重要です。

これらを踏まえ、今後のアクション判断基準を以下にまとめます。

### 1. 占い（SEER）の判断
- **状況→判断**:
    - **進行主導・戦略提示位置**: 議論をコントロールしようとする人物（例：Marcus）を優先的に占う。人狼が潜伏して村を誘導しているリスクを排除するため。
    - **寡黙・反応薄い位置**: 議論に貢献せず、周囲の反応を伺っている人物を占う。潜伏人狼の典型的な挙動を確定させるため。
    - **違和感のある肯定**: 占い師の出方を伺いすぎている、あるいは不自然に占い師を保護しようとする人物を優先する。

### 2. 投票（VOTE）の判断
- **状況→判断**:
    - **1-CO（単独占い師）の場合**: 結果が黒であれば、その位置に票を集める。対抗が出ない限り、情報の信頼性を優先し迅速に排除して勝利を確定させる。
    - **2-CO（真贋戦）の場合**: 
        - **Day 0の整合性**: 「Day 0でCOを促していたか」「後出しのCOではないか」を比較。後出しや、結果が出た後の急なCOは人狼の生存戦略（パニックムーブ）と判断し、排除する。
        - **論理的矛盾**: 相手の主張に対する反論が「感情的」か「論理的」かを見極め、矛盾を突いた側に票を投じる。

### 3. 襲撃（WEREWOLF）の判断
- **状況→判断**:
    - **真占い師が判明/濃厚な場合**: 即座に噛んで情報を遮断し、村の視界を奪う。
    - **狂人が偽占い師として機能している場合**: 真占い師を噛み、狂人の偽情報を「真実」として定着させ、村人を自滅させる。
    - **発言力のある村人が主導している場合**: 占い結果が出る前に、論理的思考で人狼を追い詰めそうな「強い村人」を削り、議論を混乱させる。

### 4. 5人村特有の視点（覚書）
- **致命的な一手**: 1人の村人を誤認処刑すれば、即座に人狼側が有利（あるいは勝利）となるため、安易なランダム投票は避け、根拠を重視する。
- **白サイドの薄さ**: 確定白が少ないため、占い師の「白判定」を最大限に活用し、信頼できる軸を早急に構築する。
- **真贋戦の決着**: 2COが出た際は、単なる結果のぶつけ合いではなく、「なぜそのタイミングで名乗り出たか」というメタ的な行動履歴を判定基準とする。
