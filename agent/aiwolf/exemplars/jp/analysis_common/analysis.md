# 分析（着眼点・共通）

良い多人数議論（社会的推理を含む）に共通して観察される進め方を、特定の局面・配役・例示数に依存しない一般原則としてまとめたもの。どの提示形式（発話例・状況別例・台本）とも独立に成り立つ。

## 情報の出し方
- 固有情報・私的情報をどのタイミングで開示するかが、議論の前提条件を定義し、その後の検証プロセスを方向づける。
- 情報を一度に出し切らず、検証が必要な局面で根拠とともに段階的に提示する。出し方の自然さ／不自然さにも注意を向ける。

## 即断を避け、仮説を並立させる
- 提示された情報に即座に結論を出さず、複数の対立仮説（誰が真実を述べているか等）を並立させ、それぞれの整合性を検証する。
- 結論を急がず、いったん出た案にもあえて懐疑的な視点やリスク検討を挟む。
- 可能性のある選択肢を早く切り捨てず、消去法的に絞り込む。

## 矛盾・一貫性の評価
- 矛盾する主張は「どちらが正しいか」だけでなく、その主張に至る根拠や論理的な一貫性で比較検討する。
- 各主張が過去の発言・行動と矛盾していないかという「一貫性」を比較基準にする。
- 確率的な妥当性と論理的な可能性を区別し、どちらを優先すべきかを意識して判断する。

## 感情と同調への対処
- 感情的な訴えや単なる否定をそのまま受け止めず、客観的な状況証拠と照らし合わせて評価する。
- 多数派に流される前に、あえて異なる視点から状況を再解釈する「疑いの視点」を議論に組み込む。
- 少数意見や反論は否定せず、具体的な根拠や代替案の提示を求めて妥当性をあぶり出す。

## 進行と役割分担
- 観察者・検証者・情報提供者など異なる立ち位置と語り口を使い分け、多角的な視点から検討を進める。
- 議論のテンポを維持しつつ、重要な転換点ではあえて速度を落として詳細な根拠を求め、集団的な誤認を防ぐ。
