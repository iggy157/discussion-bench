# 良い議論で見られた発話例

以下は、ある良い議論から抜き出した個々の発話例です（議論全体の流れは含みません）。

- どうも。占い師の方は早めに名乗り出てくれた方が効率が良いと思いますが、どうでしょう。
- 慎一さんが本物なら心強い。でも、もし偽物が混じっていたら、誰が一番不自然かを見極めたい。
- 迷わず人狼を排除しましょう。 @舞
- その反応こそが人狼らしい。陽葵さんは狂人で、人狼である舞さんを庇って村を混乱させようとしている。
- 拓海さんの言う通りだね。対抗が出た時にどう見極めるかが鍵になりそう。
- 慎一さんは舞さんが人狼だと言い、陽葵さんは舞さんが人間だと言っている。真っ向から対立しているな。
- その通りです。私は結果を正直に伝えます。嘘をつく必要はありませんから。
- もし2人出てきたら、どっちが嘘をついているか、発言の矛盾を探すしかないですよね。
- ええっ、いきなり人狼が見つかったんですか？……でも、私は占い師です。舞さんは「人間」でしたよ。
- それでは、明日の朝に占い結果を待つことにしましょう。
- 議論は十分だと思います。舞さんが人狼であるという結果を信じ、今日は彼女に投票すべきです。
- おはようございます。慎一さん、昨晩の結果はどうでしたか？
- 陽葵さんの言い方は少し感情的すぎる気がする。慎一さんは淡々と結果を伝えているし、こちらの方が信頼できそうだ。
- 陽葵さんの主張には根拠が薄い。単に「私が本物だ」と言っているだけに見える。慎一さんの結果を信じる方が合理的だ。
- 断定的なのは事実だからです。舞さんが人狼であるという結果が出た以上、迷う余地はありません。
- 信じてください！私は人間です！ @慎一
- では、私は占いに集中します。失礼します。
- おはようございます！結果が楽しみですね。
- ちょっと待ってください！私は人間です！陽葵さんが私を人間だと言ってくれたのは嬉しいけど、慎一さんの言い方はあまりに断定的すぎて怖いです。
- 慎一さんが自称したことで、明日また別の人が名乗り出るかもしれないな。
- 同意です。潜伏して後で出すより、最初から情報を共有してくれた方が村は助かるはず。
- お疲れ様でした。明日、よろしくお願いします。
- 慎一さんの占い結果を信頼する。 @舞
- 状況証拠も含めて舞さんが黒だ。 @舞
- そうだね。確証がないままに投票するのはリスクが高すぎる。
- はい。私は占い師です。昨夜、舞さんを占ったところ、結果は「人狼」でした。
- よろしくお願いします。まずは皆さんのスタンスを確認したいですね。
- まだ早いです！慎一さんを吊った方が安全かもしれません。でも、皆が慎一さんに賛成しているなら……いや、私は最後まで舞さんを信じます。
- 何を根拠にそんなことを！私は本当に占い師です。慎一さんこそ、適当な嘘をついて舞さんを吊らせようとしているだけじゃないの？
- おやすみなさい。明日の議論が楽しみです。
- 慎一さんと陽葵さんの対立だけど、舞さんが人狼である可能性が極めて高いと感じる。陽葵さんのフォローが不自然だ。
- 私は慎一さんが嘘つきだと思います。 @慎一
- よろしくです！私は村人なので、しっかり人狼を見つけ出したいと思っています。
- 陽葵さんがここまで舞さんを庇うのは、共犯関係にある証拠に近い。今日は舞さんで確定でいいだろう。 @舞
- 私は誰が占い師として出てきても、まずはその結果を信じて議論を組み立てたいです。
- 議論が早まって誰かを適当に吊る（処刑する）ことだけは避けたいところです。
- まさか、占い師が2人出たか。慎一さんと陽葵さん、どちらかが偽物ということだね。
- 舞さんの反応はどうだろう。人狼だったら、陽葵さんが自分を庇ってくれたことに乗っかって、慎一さんを攻撃するはずだ。
- 慎一さん、早急に結論を出しすぎじゃないですか？私の占い結果こそが真実です。舞さんは白（人間）ですよ。
- だって、いきなり人狼扱いされるなんて納得いかないし、必死になるのは当然じゃないですか！
- 私は明日、占い結果を持って名乗り出ます。今夜はしっかり誰を占うか決めますね。
- はい、よろしくお願いします。おやすみなさい。
- 賛成。ただ、狂人が占い師を騙る可能性もあるから、そこは慎重に判断しましょう。
- 私は本当に村人なんです！お願い、信じてください！
- 慎一さんが占い師として名乗り出るつもりなら、私も明日、自分の考えを整理して伝えます。
