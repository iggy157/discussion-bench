# 良い議論で見られた発話例

以下は、ある良い議論から抜き出した個々の発話例です（議論全体の流れは含みません）。

- ええっ！？信じられません！私は村人です。湊さんが嘘をついて、私を陥れようとしています！
- よろしく。占い師が一人だけならスムーズだけど、狂人が潜伏して攪乱してくる可能性もあるから気をつけよう。
- 陽葵さんの反論に具体性がなかった。湊さんの結果を信じて、今回は陽葵さんに票を投じる。 @陽葵
- 嘘をつく理由はありません。私は占い師として正しく結果を伝えました。陽葵さんが人狼だからこそ、そう出たんです。
- 待てよ。もし詩織さんが狂人で、湊さんが本物だとしたら、詩織さんはわざと人間である俺を白と言って信頼を得ようとしたことになるな。
- 同感。とりあえず今日は、この2人のどちらかが偽物であるという前提で夜を迎えよう。
- 詩織さんの結果は矛盾がないが、湊さんの結果が真なら陽葵さんが人狼で即終了だ。陽葵さん、何か反論はあるか？
- おっと、対抗が出たか。湊さんと詩織さんのどっちかが本物で、もう一方は人狼か狂人ってことだね。
- なるほど。詩織さんが狂人なら、湊さんの結果は信頼できる。陽葵さんが人狼である確率が極めて高い。
- 本当に冤罪なんです！湊さんが人狼で、私に罪をなすりつけて村を混乱させようとしているんだと思います！
- 陽葵さんの反応が少し慌てすぎている気がする。でも、湊さんが狂人で陽葵さんを嵌めている可能性も捨てきれない。
- 湊さんが陽葵さんを黒、詩織さんが蓮さんを白か。陽葵さん、言い分はある？
- 確かに。でも、本物が潜伏して偽物がCOするパターンもあるし、状況を整理して判断したいところ。
- 湊さんの言い方が強引に感じます。蓮さんが人間なのは確定として、湊さんと陽葵さんのどちらかが嘘をついている。
- 潜伏して人狼に消されるリスクを考えれば、早めにCOして白を確定させた方が村の利益になると判断しました。
- 陽葵さんを占いました。結果は「人狼」です。非常に心強い結果が出ましたね。
- よろしくお願いします。まずは皆さんの思考傾向を見ながら、慎重に人狼を追い詰めたいと思います。
- よろしくです！5人村なので、占い結果がかなり重要になりますね。早めに情報を出してもらえると助かります。
- 私も占い師です。湊さんの結果と私の結果を突き合わせて、どちらが真実を言っているか見極めてください。
- 詩織さんが俺を白だと言ったのはありがたいが、湊さんがいきなり陽葵さんを黒出したのは衝撃的だな。
- 感情的な議論は不要だ。明日の占い結果という客観的なデータを見て、矛盾がないか検証しよう。
- 承知しました。明日は明確な根拠を持って発言します。
- 詩織さんが後出しで出たのが気になります。湊さんの真実味を揺さぶろうとしているようにも見えますし。
- 私はまだ迷っています。ただ、陽葵さんが人狼だとしたら、私が人間であることは確定するので、村の状況は整理されます。
- そんな！お願いです、信じてください！湊さんを吊るべきです！ @湊
- 詩織さんが狂人か人狼かは分かりませんが、私は正真正銘の占い師です。明日の結果で証明します。
- おはよう。まずは占い師の二人から、昨晩の結果を教えてもらえるかな。
- 陽葵さんの反論は単なる否定で、論理的な根拠がありません。私の結果を信じて処刑すべきです。
- 迷う必要はありません。陽葵さんが人狼です。 @陽葵
- おはようございます。ドキドキしますね。どっちが本物か、しっかり見極めたいです。
- 私も準備はできています。楽しみにしていてください。
- よろしくお願いします。私は占い師ですので、明日から結果を共有して村を導きます。
- 後出しなのは認めますが、本物だからこそ名乗り出ました。湊さんの嘘を暴いてみせますよ。
- そうですね。明日、結果に食い違いが出れば、そこから絞り込めるはずです。
- じゃあ、今日はこの辺で。おやすみなさい。
- 占い師が2人出たか。ここからが本番だね。二人は誰を占ったか、明日の朝にしっかり開示してほしい。
- 私は蓮さんを占いましたが、結果は「人間」でした。湊さんの結果はあまりに都合が良すぎませんか？
- 湊さんの自信に押されましたが、論理的に考えれば陽葵さんを吊るのが今の最善手かもしれません。 @陽葵
- 湊さんが早速名乗り出たね。もし偽物だとしたら、かなり強気な出方だと思う。
- よろしくお願いします。湊さんが占い師とのことですが、対抗の方はいないのでしょうか？
