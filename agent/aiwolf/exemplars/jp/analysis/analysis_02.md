# 分析（着眼点）

- 参加者が持つ固有情報をどのタイミングで開示するかにより、議論の前提条件がどう変化し、それに合わせて他者がどう思考を切り替えるかに着目する。
- 提示された情報に対して即座に結論を出さず、複数の対立する仮説（誰が真実を述べているか等）を並立させ、それぞれの整合性を検証するプロセスを観察する。
- 感情的な反応や単なる否定ではなく、客観的な根拠や論理的な矛盾に基づいた反論が行われているか、またそれが周囲にどう受け止められたかに着目する。
- 多数派の意見に流される前に、あえて異なる視点から状況を再解釈する「疑いの視点」が議論に組み込まれているかを観察する。
- 役割や立場に応じた発言のトーンの使い分けや、議論を整理しようとする進行役のような振る舞いが、全体の合意形成にどう寄与しているかに着目する。
- 結論に至る直前まで、可能性のある選択肢を排除せず、消去法的に正解を絞り込んでいく思考プロセスの展開を観察する。
