# 分析（着眼点）

- 参加者が持つ固有情報を提示するタイミングが、議論の前提条件を定義し、その後の検証プロセスを決定づける傾向がある。
- 提示された情報に対し、即座に結論を出すのではなく、その情報の整合性や出方の不自然さを検証するステップを挟むことで、議論の精度が高まる。
- 複数の対立する主張が出た際、それぞれの主張が過去の発言や行動と矛盾していないかという「一貫性」を比較基準にする傾向がある。
- 少数意見や反論が出た際、それを単に否定せず、具体的な根拠や代替案の提示を求めることで、論理的な妥当性をあぶり出している。
- 確率的な妥当性と論理的な可能性を区別し、どちらを優先して判断すべきかを合意形成のプロセスに組み込んでいる。
- 参加者がそれぞれ「観察者」「検証者」「情報提供者」など異なる役割的な立ち位置で発言することで、多角的な視点から検討が進む傾向がある。
- 議論のテンポを維持しつつ、重要な転換点ではあえて速度を落として詳細な根拠を求めることで、集団的な誤認を防いでいる。
