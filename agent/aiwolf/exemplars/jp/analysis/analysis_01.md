# 分析（着眼点）

- 役割や権限を持つ参加者が、どのタイミングで情報を開示し、議論の方向性を提示するかに注目する。
- 提示された情報が矛盾する場合、単にどちらが正しいかではなく、その主張に至る根拠や論理的な一貫性を比較検討する傾向がある。
- 感情的な訴えや主観的な反論が出た際、それをそのまま受け止めるのではなく、客観的な状況証拠と照らし合わせて評価するプロセスが重要である。
- 複数の対立する仮説が存在する状況で、一方の主張を裏付ける状況証拠（他者の反応や不自然な擁護など）を積み上げ、確信度を高めていく流れに着目する。
- 結論を急がず、一度出た案に対してあえて懐疑的な視点を持ったり、リスクを検討したりするステップを踏むことで、判断の精度を高める傾向がある。
- 参加者がそれぞれ異なる語り口（断定的な口調、感情的な訴え、冷静な分析など）を用いることで、個々のスタンスや役割が明確に区別され、議論の構造が可視化される。
- 多数派の意見に同調するだけでなく、少数意見がどのような根拠で提示され、それがどのように論破または棄却されるかというプロセスを観察する。
