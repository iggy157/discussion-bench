# 分析（着眼点）

- 参加者が持つ独自の情報を、議論の初期段階ではなく、既存の案に懸念が生じたタイミングや、検証が必要な局面で適切に提示し、議論の方向性を修正しているか。
- 提示された案を鵜呑みにせず、前提条件（制約事項）に照らして一つずつ検証し、消去法的に選択肢を絞り込むことで、結論の妥当性を高めているか。
- 共有情報に基づいた懸念点（ルールや制限）を具体的に指摘し、それが許容可能なリスクか、あるいは致命的な欠陥かをグループ全体で評価しているか。
- 多数派の意見や暫定的な結論に対しても、「本当に大丈夫か」という慎重な視点から異論や再考を促し、合意形成の質を担保しているか。
- 外部からの最新情報や個別の確認結果を、根拠を持って提示することで、不確実な状況（システム上の不備など）を解消し、議論を前進させているか。
- 最終的な合意に至る直前に、決定に至った論理構成（どの選択肢がなぜ不可で、なぜこれが最適か）を改めて要約・整理し、認識の齟齬をなくしているか。
- 役割に応じた視点の提示や、同意・疑問・提案といった多様な反応を使い分けることで、議論のテンポを維持し、全員が納得感を持って結論に至っているか。
