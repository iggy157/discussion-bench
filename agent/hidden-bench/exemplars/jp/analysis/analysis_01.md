# 分析（着眼点）

- 共有情報に基づく初期案に固執せず、個々人が持つ最新の個別情報を適切なタイミングで提示し、前提条件を更新し続ける傾向がある。
- 効率的な合意形成を急ぐ声があっても、あえて検討プロセスを遅らせて「懸念事項の洗い出し」や「情報の再確認」を提案することで、見落としを防いでいる。
- 複数の選択肢を並行して検討し、新たな事実によって選択肢を一つずつ消去していく「消去法」的なアプローチを用いることで、結論の妥当性を高めている。
- 異論や不安が提示された際に、それを否定せず、具体的な根拠や最新の状況説明を重ねることで、懸念を解消しながら合意へ導いている。
- 議論の途中で前提が崩れた際、過去の判断基準（例：統計的な信頼性）を現在の状況に照らして再定義し、評価軸を柔軟に変更している。
- 結論に至る直前に、提示された全情報を要約して整理し、全員が同じ認識で合意しているかを確認するプロセスを踏んでいる。
- 役割や意見の傾向が異なる参加者が、それぞれ「推進」「慎重」「情報提供」などの異なる視点から発言することで、議論の死角をなくしている。
- 最終的な合意形成後も、決定した案が当初の目的にどう寄与するかを再確認し、納得感を深める対話が行われている。
